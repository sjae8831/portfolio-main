대표 성과: 시험성적서 4시간 → 5분
사용 기술: VBA, Python, React, SQL, Dashboard
강점: 단순 작업을 최소화 하기 위해 완전 자동화 시스템 구축

1. 프로젝트 소개
MES에 AI 데이터 크롤링을 학습시켜 매일 납품되는 제품의 시험성적서를 자동 발행 완성

2. 문제 정의
기존 시스템 : MES에서 수주 및 생산실적 확인 후 엑셀파일로 수동으로 기입(수작업으로 약 4시간~6시간 소요)
개선 : AI 데이터 크롤링을 사용하여 자동으로 수주관리와 생산실적 대조 후 금일 납품분의 시험성적서를 자동 발행
